# Python Programming Fundamentals – Practice Exercises

This repository contains a collection of basic Python programming exercises created to build strong foundational programming and logical reasoning skills.

The exercises focus on core concepts such as:
- Arithmetic operations
- Conditional statements
- Loops
- Functions
- Basic input/output handling

### Implemented programs include:
- Addition of two numbers  
- Factorial of a number  
- Largest of four numbers  
- Multiplication table  
- Odd or even number check  
- Palindrome check  
- Prime number check  
- Number reversal  
- Simple calculator  
- Sum of digits  

These exercises were completed as preparatory practice to strengthen Python fundamentals before progressing to data structures, machine learning, and advanced application development.

This repository is intended strictly for educational and learning purposes.
