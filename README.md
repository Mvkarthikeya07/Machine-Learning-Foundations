📦 Archived Learning Projects

## Overview
This repository serves as a consolidated archive of my foundational and learning-oriented projects developed during the early stages of my training in Computer Science, Machine Learning, and Data Science.

As my academic journey progressed toward building end-to-end systems, conducting applied research, and solving real-world problems, these projects were intentionally archived to maintain a clean, focused, and academically mature GitHub profile.

While no longer under active development, these works reflect my core understanding of algorithms, models, and programming principles that underpin my advanced projects.

## 🎯 Purpose of This Repository
- Preserve foundational and experimental projects in a single, organized location  
- Avoid cluttering my primary GitHub profile with entry-level implementations  
- Provide transparent academic evidence of my learning progression  
- Maintain reproducibility and accessibility for reviewers, evaluators, and collaborators  

This repository reflects my belief that academic growth should be curated — not erased.
